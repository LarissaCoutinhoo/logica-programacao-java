# Lógica de Programação em Java

Este repositório reúne minhas aulas, anotações e exercícios de lógica de programação utilizando Java.  
Ele funciona como um espaço de estudo contínuo, onde registro minha evolução nos fundamentos da linguagem e do raciocínio lógico aplicado à programação.

O objetivo é consolidar conceitos essenciais que servem de base para qualquer área do desenvolvimento de software.

---

## Conteúdos abordados

Entre os temas estudados estão:

- Tipos primitivos e variáveis  
- Operadores aritméticos, lógicos e relacionais  
- Estruturas condicionais (`if`, `else`, `switch`)  
- Estruturas de repetição (`for`, `while`, `do while`)  
- Entrada e saída de dados  
- Conversão de tipos  
- Expressões condicionais ternárias
- Expressões Comparativas
- Manipulação de strings  

---

## Estrutura do projeto

Os arquivos ficam organizados dentro da pasta `src`, separados por tema. 
Cada arquivo representa um assunto específico da lógica de programação em Java.

---

## Objetivo do repositório

- Fixar a base de lógica de programação  
- Praticar Java de forma estruturada  
- Criar um histórico de aprendizado  
- Servir como material de consulta futura  
- Demonstrar organização e evolução técnica  

---

## Em constante evolução :)

Novos exercícios e conteúdos serão adicionados conforme o avanço nos estudos.
